Examples
########

This page provides two examples of how the package
:doc:`kgw <package_references>`
can be used to define and run workflows for processing knowledge graphs.
It assumes that you are using a Python 3 interpreter and
that you have installed the package as explained in the
:doc:`Installation Guide <installation>`.
